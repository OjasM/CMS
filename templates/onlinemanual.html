Discussion Forum


When a user clicks on the discussion link from navigation-bar or home-page s/he will be redirected to discussion page
It will show the present going-on discussions and each discussion will have upvote,time created,created-by,view comments,no of comments functionality, and displays four additional features:

1:Search

In this feature, user can search the discussion topics by title name or it can search the topics created by a particular person



2:My Discussion

In this user can view the topics created by him and can delete them too.



3:Popular Discussion

In this feature , user can see the popular discussions based upon no of upvotes and comments.




4:Adding Comments to a particular topic

To add comments to a particular topic user has to open the particular topic and click "comment" to add a comment.






Polls


When a user clicks on the Polls link from navigation-bar or home-page s/he will be redirected to polls page
It will show the present going-on polls and each poll will have time created,created-by,choices "Yes or No or Can't Say" functionality, and displays three additional features:

1:Search

In this feature, user can search the polls by name or it can search the polls created by a particular person



2:My Polls

In this user can view the polls created by him and can delete them too.



3:Popular Polls

In this feature , user can see the popular polls based upon no of responses .





ATTENDANCE

For Teacher

MAIN PAGE:



When a teacher clicks on the attendance tab, he/she is redirected to this page. This page shows all the courses which the teacher is teaching. Each course shows 3 options:

1: View attendance

On clicking this button the teacher can view the dates on which attendance for that course was taken.

2: Take Today's Attendance

On clicking this button the teacher can take attendance for any particular date or todays date.

3: View attendance report

On clicking this button the teacher can view the list of percentage wise attendances of all the students enrolled in that course


VIEW ATTENDANCE PAGE

­­On the view attendance page, the teacher can view different dates on which attendance was taken sorted date wise. On clicking any date, the details for that day’s attendance will be shown.

EDIT ATTENDANCE PAGE

On this page the teacher can view attendance for students for that date. The teacher has the option to edit that attendance also by making suitable changes and clicking on the edit attendance button. Also the summarised data for that date like total students, present students and absent students are displayed.

TAKE ATTENDANCE

On this page the teacher enters the date for which he wants to enter the attendance. He then takes the corresponding attendance and clicks on the save attendance button to save the attendance.

ATTENDANCE LIST PAGE

On this page the teacher can view percentage wise attendance for students of that course.

For Students



When a student clicks on the attendance tab, he/she is redirected to this page. This page shows the courses in which the student is currently enrolled. Each course has view attendance report button by clicking on which the students can view their summary of attendance in that course

SUBJECT SELECTED PAGE

On this page the student gets various tools to analyse his attendance :

1: Total No of Classes

– this is the total number of classes on which attendance has been taken until now.

2: Present Classes

– the total days on which he was present.

3: Absent Classes

– the total days on which he was absent.

4: Percentage Attendance

– attendance percentage calculated on the basis of total, present classes.

5: Projected No of Classes

– no of classes the student is likely going to attend by attending classes at current rate.

6: Maximum Possible Attendance

– the maximum attendance rate that the student can gain by attending all the classes from now on.

Security Features


Course Management System is well-equipped with two kind of security functionalities, forgot password and change password.

1:ForgotPassword

If user forgets his password, it can change his password by clicking on the "forgot password " link and submitting his verified email





2:ChangePassword

If user feels vulnerable of his/her password, can go to "change password " link and change password as and when required.
